==================================================
  SERVICE TIMER — TV SETUP INSTRUCTIONS
==================================================

STEP 1 — On the Xiaomi TV Stick 4K
-----------------------------------
Open any web browser on the TV (e.g. Google Chrome,
Firefox, or the built-in browser).

STEP 2 — Type this URL into the browser address bar:
------------------------------------------------------

  https://charleskcoffie-a11y.github.io/SERVICE-MANAGEMENT/?mode=tv

  (Type it exactly as shown, including the ?mode=tv at the end)

STEP 3 — What you will see on the TV:
--------------------------------------
- The current time (large clock)
- The active service item currently running
- The countdown timer
- No buttons or controls — display only

The TV updates automatically in real-time whenever
the admin makes changes on the laptop.

If you are opening the app for St Pauls Baltemore,
use the same base URL with the society name attached:

  https://charleskcoffie-a11y.github.io/SERVICE-MANAGEMENT/?mode=tv&society=St%20Pauls%20Baltemore

For GMCT, you can keep using the original link,
or use the explicit society version:

  https://charleskcoffie-a11y.github.io/SERVICE-MANAGEMENT/?mode=tv&society=GMCT
